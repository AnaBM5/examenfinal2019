Vamos a desplegar servicios en amazon con software libre.

1. Construir un portal web que cumpla una funcionalidad clave con software libre (el de su preferencia):
Recomendado revisar esto: https://rapidapi.com/collection/list-of-free-apis

 - Generar un password aleatorio con caracteres alfanuméricos.
 - Decir la hora de alguna parte de Asia.
 - Traer un gif aleatorio de Giphy o un servicio similar.
 - El clima del día siguiente. (con cualquier servicio).
 - La ubicación actual del usuario.
 - Generar un nombre aleatorio que tenga sentido.
 - El precio actual del dólar.
 - Una cita célebre aleatoria.
 - Mostrar una imagen famosa aleatoria.
 - Integrar todo lo anterior en un menú o directorio.

2. El portal debe además incluir nombres y el logo de la upb.

3. Hacer seguimiento al portal implementado por medio de este repositorio en una rama aparte con nombre: (nombre_nombre).

4. Desplegar el servicio desde Github a AWS EC2 con git. Para esto hay una guía en https://gist.github.com/noelboss/3fe13927025b89757f8fb12e9066f2fa
También se puede utilizar directamente el script BASH en este repositorio llamado post-recieve.sh

5. (BONUS) Subir a Amazon S3 una copia de la licencia que utilizarían diferente a la MPL original de este repositorio pero compatible con ella. De lo contrario subirla a la rama.
